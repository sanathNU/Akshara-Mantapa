# ಅಕ್ಷರ ಮಂಟಪ

![ಅಕ್ಷರ ಮಂಟಪ ಬ್ಯಾನರ್](frontend/static/main-picture.png)

ಕನ್ನಡಕ್ಕಾಗಿ ಬ್ಯಾಬೆಲ್ ಗ್ರಂಥಾಲಯ.

ಜಾರ್ಜ್ ಲೂಯಿಸ್ ಬೋರ್ಗೆಸ್ ಅವರ "ದಿ ಲೈಬ್ರರಿ ಆಫ್ ಬ್ಯಾಬೆಲ್" ಕಿರು ಕಥೆಯಿಂದ ಸ್ಫೂರ್ತಿ ಪಡೆದ, ಕನ್ನಡ ಪಠ್ಯದ ಎಲ್ಲಾ ಸಂಭವನೀಯ ಸಂಯೋಜನೆಗಳನ್ನು ಒಳಗೊಂಡ ಅನಂತ ಗ್ರಂಥಾಲಯ.

## ಪರಿಕಲ್ಪನೆ

ಈ ಯೋಜನೆಯು **ಏಕೈಕ, ಸುಂದರವಾದ ದ್ವಿಮುಖ ಮ್ಯಾಪಿಂಗ್** ಬಳಸಿ ಕನ್ನಡ ಭಾಷೆಗಾಗಿ ಬ್ಯಾಬೆಲ್ ಗ್ರಂಥಾಲಯದ ಡಿಜಿಟಲ್ ಆವೃತ್ತಿಯನ್ನು ಅಳವಡಿಸುತ್ತದೆ. ೪೦೦ ಅಕ್ಷರಗುಚ್ಛಗಳ ಪ್ರತಿಯೊಂದು ಸಂಭವನೀಯ ಕನ್ನಡ ಪಠ್ಯವು ನಿಖರವಾಗಿ ಒಂದು ವಿಳಾಸದಲ್ಲಿ ಅಸ್ತಿತ್ವದಲ್ಲಿದೆ, ಮತ್ತು ಪ್ರತಿ ವಿಳಾಸವು ನಿಖರವಾಗಿ ಒಂದು ಪುಟವನ್ನು ಉತ್ಪಾದಿಸುತ್ತದೆ. ವ್ಯವಸ್ಥೆಯು ಸಂಪೂರ್ಣವಾಗಿ ವಿಲೋಮಿಸಬಲ್ಲದು—ನೀವು ಯಾವುದೇ ಪಠ್ಯವನ್ನು ಹುಡುಕಬಹುದು ಮತ್ತು ಅದರ ಅನನ್ಯ ಸ್ಥಳವನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು, ಮತ್ತು ಪ್ರತಿ ಸ್ಥಳವು ನಿರ್ಣಾಯಕವಾಗಿ ಅದರ ವಿಷಯವನ್ನು ಉತ್ಪಾದಿಸುತ್ತದೆ.

## ವೈಶಿಷ್ಟ್ಯಗಳು

- **ಯಾದೃಚ್ಛಿಕ ಪುಟಗಳನ್ನು ಅನ್ವೇಷಿಸಿ**: ಅನಂತ ಗ್ರಂಥಾಲಯದಿಂದ ಯಾದೃಚ್ಛಿಕ ಪುಟಗಳನ್ನು ಅನ್ವೇಷಿಸಿ
- **ಪಠ್ಯಕ್ಕಾಗಿ ಹುಡುಕಿ**: ಯಾವುದೇ ಕನ್ನಡ ಪಠ್ಯದ ನಿಖರ ಸ್ಥಳವನ್ನು ತಕ್ಷಣ ಹುಡುಕಿ (ನೀಲಿ ಹೈಲೈಟ್)
- **ಮತ್ತೆ ಹುಡುಕಿ**: ಪುಟಗಳಲ್ಲಿ ಯಾದೃಚ್ಛಿಕ ಸ್ಥಾನಗಳಲ್ಲಿ ಪಠ್ಯವನ್ನು ಹುಡುಕಿ (ಹಳದಿ ಹೈಲೈಟ್)
- **ವಿಳಾಸದ ಮೂಲಕ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿ**: ಶ್ರೇಣೀಕೃತ ಅಥವಾ ಹೆಕ್ಸ್ ವಿಳಾಸಗಳನ್ನು ಬಳಸಿ ಯಾವುದೇ ಪುಟಕ್ಕೆ ಜಿಗಿಯಿರಿ
- **ಪುಟ ನ್ಯಾವಿಗೇಶನ್**: ಮುಂದಿನ/ಹಿಂದಿನ ಬಟನ್‌ಗಳೊಂದಿಗೆ ಮೃದುವಾಗಿ ಪುಟಗಳನ್ನು ಬ್ರೌಸ್ ಮಾಡಿ
- **ಶ್ರೇಣೀಕೃತ ಪ್ರದರ್ಶನ**: ವಿಳಾಸಗಳು `ಮಂದಿರ.ಗೋಡೆ.ಪಟ್ಟಿ.ಪುಸ್ತಕ.ಪುಟ` ರೂಪದಲ್ಲಿ ತೋರಿಸಲ್ಪಡುತ್ತವೆ
- **ಕನ್ನಡದಲ್ಲಿ ಮಂದಿರ**: ಚಿಕ್ಕ ವಿಳಾಸಗಳಿಗೆ ಕೋಣೆ ಗುರುತಿಸುವಿಕೆಗಳು ಕನ್ನಡ ಲಿಪಿಯಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲ್ಪಡುತ್ತವೆ
- **ದ್ವಿಮುಖ ಮ್ಯಾಪಿಂಗ್**: ಗುಣಾಕಾರ ವಿಲೋಮ ಮಾಡ್ಯುಲರ್ ಅಂಕಗಣಿತ ಬಳಸಿ ಏಕ ವಿಲೋಮಿಸಬಲ್ಲ ವ್ಯವಸ್ಥೆ
- **ಸಮೃದ್ಧ ಕನ್ನಡ ಲಿಪಿ**: **೫೭,೩೨೪ ಅಕ್ಷರಗುಚ್ಛಗಳನ್ನು** ಒಳಗೊಂಡಿದೆ:
  - ವ್ಯಂಜನಗಳು, ಸ್ವರಗಳು, ಮತ್ತು ಅವುಗಳ ಸಂಯೋಜನೆಗಳು
  - ಮಾತ್ರೆಗಳು (ಸ್ವರ ಚಿಹ್ನೆಗಳು) ಮತ್ತು ಮಾರ್ಪಾಡುಗಳು (ಂ, ಃ)
  - ಒತ್ತಕ್ಷರಗಳು (ವ್ಯಂಜನ ಗುಚ್ಛಗಳು ಉದಾ: ಕ್ಷ)
  - ವಿರಾಮ ಚಿಹ್ನೆಗಳು ಮತ್ತು ಅಂತರಗಳು
- **ಕನಿಷ್ಠ ವಿನ್ಯಾಸ**: ಸೆರಿಫ್ ಅಕ್ಷರಶೈಲಿಯೊಂದಿಗೆ ಶುದ್ಧ, ವಿದ್ವತ್ಪೂರ್ಣ ಸೌಂದರ್ಯ
- **ದ್ವಿ ಮೋಡ್**: ಅಭಿವೃದ್ಧಿಗೆ HTTP API, ಉತ್ಪಾದನೆಗೆ WASM (GitHub Pages)

## ಗಣಿತ

### ದ್ವಿಮುಖ ಸೂತ್ರ

ಪ್ರತಿ ಪುಟದ ವಿಷಯವು ಅನನ್ಯ ವಿಳಾಸಕ್ಕೆ ಮ್ಯಾಪ್ ಆಗುತ್ತದೆ:

```
content_num = Σ (cluster_index[i] × 56028^i) for i in 0..400
address = (content_num × C) mod N
content_num = (address × I) mod N

ಇಲ್ಲಿ:
- N = 56028^400 (ಮಾಡ್ಯುಲಸ್, ಒಟ್ಟು ಸಂಭವನೀಯ ಪುಟಗಳ ಸಂಖ್ಯೆ)
- C = ಸಹ-ಅವಿಭಾಜ್ಯ ಗುಣಕ
- I = C^(-1) mod N (C ಯ ಮಾಡ್ಯುಲರ್ ವಿಲೋಮ)
```

ಇದು ವಿಷಯ ಮತ್ತು ವಿಳಾಸಗಳ ನಡುವೆ ಪರಿಪೂರ್ಣ ಒಂದು-ಒಂದು ಮ್ಯಾಪಿಂಗ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ.

### ಸ್ಥಿರಾಂಕಗಳು (ಬೋರ್ಗೆಸ್-ನಿಷ್ಠ)

```
ವರ್ಣಮಾಲೆ_ಗಾತ್ರ       = ೫೭,೩೨೪ ಅಕ್ಷರಗುಚ್ಛಗಳು
ಪುಟಕ್ಕೆ_ಗುಚ್ಛಗಳು      = ೪೦೦
ಪುಸ್ತಕಕ್ಕೆ_ಪುಟಗಳು     = ೪೧೦
ಪಟ್ಟಿಗೆ_ಪುಸ್ತಕಗಳು     = ೩೨
ಗೋಡೆಗೆ_ಪಟ್ಟಿಗಳು       = ೫
ಮಂದಿರಕ್ಕೆ_ಗೋಡೆಗಳು     = ೪
```

### ವಿಳಾಸ ಜಾಗ

```
ಒಟ್ಟು ಪುಟಗಳು    = ೫೭,೩೨೪^೪೦೦ ≈ ೧೦^೧,೮೯೯
ವಿಳಾಸ ಬಿಟ್‌ಗಳು   ≈ ೬,೩೦೮ ಬಿಟ್‌ಗಳು (~೭೮೯ ಬೈಟ್‌ಗಳು, ~೧,೫೭೮ ಹೆಕ್ಸ್ ಅಕ್ಷರಗಳು)
```

## ವಿಳಾಸ ಸ್ವರೂಪ

### ಕಚ್ಚಾ ಹೆಕ್ಸ್ ವಿಳಾಸ
```
93cebf0ea1c7096fe3de06fd119faec4c4d549bda55708577c... (~೧,೫೭೮ ಅಕ್ಷರಗಳು)
```

### ಶ್ರೇಣೀಕೃತ ಪ್ರದರ್ಶನ
```
ಸ್ವರೂಪ: ಮಂದಿರ.ಗೋಡೆ.ಪಟ್ಟಿ.ಪುಸ್ತಕ.ಪುಟ
ಉದಾಹರಣೆ: 24ea75265...849e4ebd.4.4.23.325

ಘಟಕಗಳು:
- ಮಂದಿರ: ಕೋಣೆ ಗುರುತಿಸುವಿಕೆ ಹೆಕ್ಸ್‌ನಲ್ಲಿ (~೧,೫೬೦ ಅಕ್ಷರಗಳು)
- ಗೋಡೆ: ಗೋಡೆ ಸಂಖ್ಯೆ (೧-೪)
- ಪಟ್ಟಿ: ಪಟ್ಟಿ ಸಂಖ್ಯೆ (೧-೫)
- ಪುಸ್ತಕ: ಪುಸ್ತಕ ಸಂಖ್ಯೆ (೧-೩೨)
- ಪುಟ: ಪುಟ ಸಂಖ್ಯೆ (೧-೪೧೦)
```

### ಕನ್ನಡದಲ್ಲಿ ಮಂದಿರ
ಚಿಕ್ಕ ವಿಳಾಸಗಳಿಗೆ (< ೧೦,೦೦೦ ಬಿಟ್‌ಗಳು), ಮಂದಿರ (ಕೋಣೆ ಹೆಸರು) ಕನ್ನಡ ಅಕ್ಷರಗುಚ್ಛಗಳಾಗಿ ಪ್ರದರ್ಶಿಸಲ್ಪಡುತ್ತದೆ:
```
ಕವಿರಾಜಮಾರ್ಗದಲ್ಲಿಯೇಕನ್ನಡನಾಡಿನ...
```

## ತಂತ್ರಜ್ಞಾನ ಸ್ಟ್ಯಾಕ್

### ಬ್ಯಾಕೆಂಡ್
- **Rust** - ಉನ್ನತ-ಕಾರ್ಯಕ್ಷಮತೆಯ, ಮೆಮೊರಿ-ಸುರಕ್ಷಿತ ಸಿಸ್ಟಮ್ಸ್ ಪ್ರೋಗ್ರಾಮಿಂಗ್
- **Axum** - ಆಧುನಿಕ ಅಸಿಂಕ್ ವೆಬ್ ಫ್ರೇಮ್‌ವರ್ಕ್
- **num-bigint** - ೬,೩೦೮-ಬಿಟ್ ಸಂಖ್ಯೆಗಳಿಗೆ ಅನಿಯಮಿತ ನಿಖರತೆಯ ಅಂಕಗಣಿತ
- **num-integer** - ಪೂರ್ಣಾಂಕ ಕಾರ್ಯಾಚರಣೆಗಳು (GCD, ಶೇಷದೊಂದಿಗೆ ವಿಭಜನೆ)
- **Serde** - JSON ಧಾರಾವಾಹಿಕರಣ
- **Tower-HTTP** - CORS ಮತ್ತು ಮಿಡಲ್‌ವೇರ್
- **wasm-bindgen** - ಬ್ರೌಸರ್ ನಿಯೋಜನೆಗೆ WASM ಬೈಂಡಿಂಗ್‌ಗಳು

### ಫ್ರಂಟೆಂಡ್
- **SvelteKit** - ಆಧುನಿಕ, ಪ್ರತಿಕ್ರಿಯಾತ್ಮಕ UI ಫ್ರೇಮ್‌ವರ್ಕ್
- **TypeScript** - ಪ್ರಕಾರ-ಸುರಕ್ಷಿತ ಅಭಿವೃದ್ಧಿ
- **Vite** - ಅತ್ಯಂತ ವೇಗದ ಡೆವ್ ಸರ್ವರ್
- **Noto Sans Kannada** - ಸರಿಯಾದ ಕನ್ನಡ ಫಾಂಟ್ ರೆಂಡರಿಂಗ್

## ಅಳವಡಿಕೆ ಮತ್ತು ಸ್ಥಾಪನೆ

### ಪೂರ್ವಾಪೇಕ್ಷಿತಗಳು

- **Rust** (1.70+): [rustup.rs](https://rustup.rs) ನಿಂದ ಸ್ಥಾಪಿಸಿ
- **Node.js** (18+): [nodejs.org](https://nodejs.org) ನಿಂದ ಸ್ಥಾಪಿಸಿ
- **wasm-pack**: `cargo install wasm-pack` ನಿಂದ ಸ್ಥಾಪಿಸಿ

### ಅಭಿವೃದ್ಧಿ ಮೋಡ್ (HTTP API)

#### ಬ್ಯಾಕೆಂಡ್ ಸೆಟಪ್

1. ಬ್ಯಾಕೆಂಡ್ ಡೈರೆಕ್ಟರಿಗೆ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿ:
   ```bash
   cd backend
   ```

2. ಸರ್ವರ್ ಅನ್ನು ನಿರ್ಮಿಸಿ ಮತ್ತು ಚಲಾಯಿಸಿ:
   ```bash
   cargo run --bin server --release
   ```

   ಬ್ಯಾಕೆಂಡ್ `http://127.0.0.1:3000` ನಲ್ಲಿ ಪ್ರಾರಂಭವಾಗುತ್ತದೆ

   ದ್ವಿಮುಖ ಸ್ಥಿರಾಂಕಗಳನ್ನು (C ಮತ್ತು I) ಲೆಕ್ಕಹಾಕಲು ಮೊದಲ ಪ್ರಾರಂಭಕ್ಕೆ ಕೆಲವು ಸೆಕೆಂಡುಗಳು ತೆಗೆದುಕೊಳ್ಳುತ್ತದೆ.

#### ಫ್ರಂಟೆಂಡ್ ಸೆಟಪ್

1. ಫ್ರಂಟೆಂಡ್ ಡೈರೆಕ್ಟರಿಗೆ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿ:
   ```bash
   cd frontend
   ```

2. ಅವಲಂಬನೆಗಳನ್ನು ಸ್ಥಾಪಿಸಿ:
   ```bash
   npm install
   ```

3. ಅಭಿವೃದ್ಧಿ ಸರ್ವರ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸಿ:
   ```bash
   npm run dev
   ```

   ಫ್ರಂಟೆಂಡ್ `http://localhost:5173` ನಲ್ಲಿ ಪ್ರಾರಂಭವಾಗುತ್ತದೆ

### ಉತ್ಪಾದನೆ ಮೋಡ್ (WASM)

ಸ್ಥಿರ ನಿಯೋಜನೆಗೆ (ಉದಾ: GitHub Pages), ಫ್ರಂಟೆಂಡ್ HTTP API ಬದಲಿಗೆ WASM ಬಳಸುತ್ತದೆ.

1. WASM ನಿರ್ಮಿಸಿ:
   ```bash
   cd backend
   wasm-pack build --target web --features wasm
   ```

2. WASM ಫೈಲ್‌ಗಳನ್ನು ಫ್ರಂಟೆಂಡ್‌ಗೆ ನಕಲಿಸಿ:
   ```bash
   cp -r pkg/* ../frontend/src/lib/wasm/
   ```

3. ಫ್ರಂಟೆಂಡ್ ನಿರ್ಮಿಸಿ:
   ```bash
   cd frontend
   npm run build
   ```

4. `build/` ಫೋಲ್ಡರ್ ಅನ್ನು ಯಾವುದೇ ಸ್ಥಿರ ಹೋಸ್ಟ್‌ಗೆ ನಿಯೋಜಿಸಬಹುದು.

## ಹೇಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ

### ದ್ವಿಮುಖ ಮ್ಯಾಪಿಂಗ್

ವ್ಯವಸ್ಥೆಯು ಪರಿಪೂರ್ಣ ದ್ವಿಮುಖವನ್ನು ರಚಿಸಲು **ಗುಣಾಕಾರ ವಿಲೋಮ ಮಾಡ್ಯುಲರ್ ಅಂಕಗಣಿತ** ಬಳಸುತ್ತದೆ:

1. **ಹುಡುಕಾಟ (ಪಠ್ಯ → ವಿಳಾಸ)**:
   - ಇನ್‌ಪುಟ್ ಪಠ್ಯ ತೆಗೆದುಕೊಳ್ಳಿ: `"ಕನ್ನಡ"`
   - ಅಕ್ಷರಗುಚ್ಛಗಳಾಗಿ ವಿಭಜಿಸಿ: `["ಕ", "ನ್", "ನ", "ಡ"]`
   - ಪ್ರತಿ ಗುಚ್ಛವನ್ನು ಅದರ ವರ್ಣಮಾಲೆ ಸೂಚ್ಯಂಕಕ್ಕೆ ಪರಿವರ್ತಿಸಿ
   - ಅಂತರಗಳೊಂದಿಗೆ ೪೦೦ ಗುಚ್ಛಗಳಿಗೆ ಪ್ಯಾಡ್ ಮಾಡಿ (ಸೂಚ್ಯಂಕ ೦)
   - ಬೇಸ್-೫೭,೩೨೪ ಸಂಖ್ಯೆಯಾಗಿ ಪರಿಗಣಿಸಿ: `content_num = Σ (index[i] × 56028^i)`
   - ದ್ವಿಮುಖ ಅನ್ವಯಿಸಿ: `address = (content_num × C) mod N`
   - ಹೆಕ್ಸ್ ಮತ್ತು ಶ್ರೇಣೀಕೃತ ಸ್ವರೂಪಕ್ಕೆ ಪರಿವರ್ತಿಸಿ

2. **ಬ್ರೌಸ್ (ವಿಳಾಸ → ಪಠ್ಯ)**:
   - ಹೆಕ್ಸ್ ವಿಳಾಸ ತೆಗೆದುಕೊಳ್ಳಿ
   - ವಿಲೋಮ ಅನ್ವಯಿಸಿ: `content_num = (address × I) mod N`
   - ಬೇಸ್-೫೭,೩೨೪ ಸೂಚ್ಯಂಕಗಳಿಗೆ ಮರಳಿ ಪರಿವರ್ತಿಸಿ
   - ಸೂಚ್ಯಂಕಗಳನ್ನು ಅಕ್ಷರಗುಚ್ಛಗಳಿಗೆ ಮ್ಯಾಪ್ ಮಾಡಿ
   - ಸಾಲಿಗೆ ೨೫ ಗುಚ್ಛಗಳಾಗಿ ಫಾರ್ಮ್ಯಾಟ್ ಮಾಡಿ

### ಕನ್ನಡ ಅಕ್ಷರ ಸಮುಚ್ಚಯ

ವರ್ಣಮಾಲೆಯು ಎಲ್ಲಾ ಮಾನ್ಯ ಕನ್ನಡ ಅಕ್ಷರಗುಚ್ಛಗಳನ್ನು ವ್ಯವಸ್ಥಿತವಾಗಿ ಉತ್ಪಾದಿಸುತ್ತದೆ:
- ವಿರಾಮ ಚಿಹ್ನೆಗಳು ಮತ್ತು ಅಂತರಗಳು
- ಐಚ್ಛಿಕ ಮಾರ್ಪಾಡುಗಳೊಂದಿಗೆ ಸ್ವತಂತ್ರ ಸ್ವರಗಳು
- ಸರಳ ವ್ಯಂಜನಗಳು
- ಮಾತ್ರೆಗಳು ಮತ್ತು ಮಾರ್ಪಾಡುಗಳೊಂದಿಗೆ ವ್ಯಂಜನಗಳು
- ಮೃತ ವ್ಯಂಜನಗಳು (ಹಲಂತದೊಂದಿಗೆ)
- ಎಲ್ಲಾ ಸಂಯೋಜನೆಗಳೊಂದಿಗೆ ಎರಡು-ವ್ಯಂಜನ ಒತ್ತಕ್ಷರಗಳು

ಇದು ನಿಖರವಾಗಿ **೫೭,೩೨೪ ಅನನ್ಯ ಗುಚ್ಛಗಳನ್ನು** ಉತ್ಪಾದಿಸುತ್ತದೆ.

### ಪುಟ ವಿಶೇಷಣಗಳು

- **ಪುಟಕ್ಕೆ ಗುಚ್ಛಗಳು**: ೪೦೦
- **ಫಾರ್ಮ್ಯಾಟ್ ಮಾಡಿದ ಪ್ರದರ್ಶನ**: ಸಾಲಿಗೆ ೨೫ ಗುಚ್ಛಗಳು, ೧೬ ಸಾಲುಗಳು

## ವಿನ್ಯಾಸ ತತ್ವಶಾಸ್ತ್ರ

ಇಂಟರ್‌ಫೇಸ್ ಕನಿಷ್ಠ, ವಿದ್ವತ್ಪೂರ್ಣ ಸೌಂದರ್ಯವನ್ನು ಅನುಸರಿಸುತ್ತದೆ:
- ಮುಖ್ಯ ಪಠ್ಯಕ್ಕೆ ಸೆರಿಫ್ ಅಕ್ಷರಶೈಲಿ (Georgia)
- ತಾಂತ್ರಿಕ ವಿವರಗಳಿಗೆ ಮೊನೊಸ್ಪೇಸ್ ಫಾಂಟ್‌ಗಳು (Courier)
- ಹೆಚ್ಚಿನ ವ್ಯತಿರಿಕ್ತತೆ (ಬಿಳಿಯ ಮೇಲೆ ಕಪ್ಪು)
- ಉದಾರ ಬಿಳಿ ಜಾಗ
- ಶುದ್ಧ ಅಂಚುಗಳು ಮತ್ತು ಸೂಕ್ಷ್ಮ ವಿಭಜಕಗಳು
- ಲೋಡಿಂಗ್ ಸೂಚಕಗಳೊಂದಿಗೆ ಮೃದುವಾದ ಪುಟ ಪರಿವರ್ತನೆಗಳು
- ಓದುವಿಕೆ ಮತ್ತು ವಿಷಯದ ಮೇಲೆ ಒತ್ತು
- ಸರಿಯಾದ ಫಾಂಟ್ ರೆಂಡರಿಂಗ್‌ನೊಂದಿಗೆ ಕನ್ನಡ-ಮೊದಲ ವಿನ್ಯಾಸ

## ಕಾರ್ಯಕ್ಷಮತೆ ಗುಣಲಕ್ಷಣಗಳು

- **ಪಠ್ಯ ಉತ್ಪಾದನೆ**: O(n²) ಇಲ್ಲಿ n ≈ ೬,೩೦೦ ಬಿಟ್‌ಗಳು - ಅತ್ಯಂತ ವೇಗವಾಗಿ!
- **ಹುಡುಕಾಟ**: O(n²) - ಯಾವುದೇ ಪ್ರಶ್ನೆಗೆ ತಕ್ಷಣ
- **ವಿಳಾಸ ಪಾರ್ಸಿಂಗ್**: O(n) - ದಕ್ಷ
- **ಡೇಟಾಬೇಸ್ ಇಲ್ಲ**: ಎಲ್ಲವೂ ನಿರ್ಣಾಯಕ ಮತ್ತು ಬೇಡಿಕೆಯ ಮೇಲೆ ಲೆಕ್ಕಹಾಕಲಾಗುತ್ತದೆ
- **ಮೊದಲ ಪ್ರಾರಂಭ**: ~೫ ಸೆಕೆಂಡುಗಳು ದ್ವಿಮುಖ ಸ್ಥಿರಾಂಕಗಳನ್ನು ಲೆಕ್ಕಹಾಕಲು
- **ನಂತರದ ವಿನಂತಿಗಳು**: ಮಿಲಿಸೆಕೆಂಡುಗಳು

## ಭವಿಷ್ಯದ ವರ್ಧನೆಗಳು

1. **ಸ್ಥಿರಾಂಕಗಳನ್ನು ಪೂರ್ವ-ಲೆಕ್ಕಹಾಕಿ** - ತಕ್ಷಣ ಪ್ರಾರಂಭಕ್ಕೆ C ಮತ್ತು I ಅನ್ನು ಫೈಲ್‌ಗಳಲ್ಲಿ ಉಳಿಸಿ
2. **ಮಂದಿರ ಕ್ಯಾಶಿಂಗ್** - ಕನ್ನಡ ಮಂದಿರ ಪರಿವರ್ತನೆಗಳನ್ನು ಕ್ಯಾಶ್ ಮಾಡಿ
3. **ದರ ಮಿತಿ** - API ಅನ್ನು ದುರುಪಯೋಗದಿಂದ ರಕ್ಷಿಸಿ
4. **LRU ಕ್ಯಾಶ್** - ಆಗಾಗ್ಗೆ ಪ್ರವೇಶಿಸಿದ ಪುಟಗಳನ್ನು ಕ್ಯಾಶ್ ಮಾಡಿ
5. **ಬಹು ಲಿಪಿಗಳು** - ಇತರ ಭಾರತೀಯ ಲಿಪಿಗಳಿಗೆ ಬೆಂಬಲ (ತೆಲುಗು, ತಮಿಳು, ಮಲಯಾಳಂ)
6. **ಕತ್ತಲೆ ಮೋಡ್** - ವಿವಿಧ ಬೆಳಕಿನ ಪರಿಸ್ಥಿತಿಗಳಿಗೆ ಥೀಮ್ ಟಾಗಲ್
7. **ಪುಟಗಳನ್ನು ರಫ್ತು ಮಾಡಿ** - PDF ಅಥವಾ ಪಠ್ಯವಾಗಿ ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ
8. **ಸುಧಾರಿತ ಹುಡುಕಾಟ** - Regex ಅಥವಾ ಮಾದರಿ-ಆಧಾರಿತ ಹುಡುಕಾಟ
9. **ಬುಕ್‌ಮಾರ್ಕ್‌ಗಳು** - ಮೆಚ್ಚಿನ ಪುಟಗಳನ್ನು ಅಥವಾ ಸ್ಥಳಗಳನ್ನು ಉಳಿಸಿ
10. **ಹಂಚಿಕೆ ಲಿಂಕ್‌ಗಳು** - ನಿರ್ದಿಷ್ಟ ಪುಟಗಳಿಗೆ ಹಂಚಿಕೊಳ್ಳಬಹುದಾದ URL ಗಳನ್ನು ರಚಿಸಿ

## ತತ್ವಶಾಸ್ತ್ರ

> "ವಿಶ್ವವು (ಇತರರು ಗ್ರಂಥಾಲಯ ಎಂದು ಕರೆಯುತ್ತಾರೆ) ಅನಿಶ್ಚಿತ
> ಮತ್ತು ಬಹುಶಃ ಅನಂತ ಸಂಖ್ಯೆಯ ಷಡ್ಭುಜಾಕಾರದ ಗ್ಯಾಲರಿಗಳಿಂದ ಕೂಡಿದೆ..."
>
> — ಜಾರ್ಜ್ ಲೂಯಿಸ್ ಬೋರ್ಗೆಸ್, "ದಿ ಲೈಬ್ರರಿ ಆಫ್ ಬ್ಯಾಬೆಲ್"

ಈ ಅಳವಡಿಕೆಯು ನಿರ್ಣಾಯಕ ಅಲ್ಗಾರಿದಮ್‌ಗಳು ಮತ್ತು ಸೊಗಸಾದ ಗಣಿತದೊಂದಿಗೆ, ಪ್ರತಿ ಸಂಭವನೀಯ ಪಠ್ಯವು ನಿರ್ದಿಷ್ಟ, ಲೆಕ್ಕಹಾಕಬಹುದಾದ ಸ್ಥಳದಲ್ಲಿ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಅನಂತ, ಪುನರುತ್ಪಾದಿಸಬಹುದಾದ ಜಾಗವನ್ನು ನಾವು ರಚಿಸಬಹುದು ಎಂದು ಸಾಬೀತುಪಡಿಸುತ್ತದೆ.

೫೭,೩೨೪ ಅಕ್ಷರಗುಚ್ಛಗಳೊಂದಿಗೆ ಕನ್ನಡ ಲಿಪಿಯ ಬಳಕೆಯು ಮೂಲಕ್ಕಿಂತ (29^3200 vs 56028^400) ಬಹಳ ದೊಡ್ಡ ಜಾಗವನ್ನು ರಚಿಸುತ್ತದೆ, ಆದರೂ ದ್ವಿಮುಖ ಮ್ಯಾಪಿಂಗ್ ಮೂಲಕ ಸಂಪೂರ್ಣವಾಗಿ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಬಹುದಾಗಿ ಉಳಿಯುತ್ತದೆ.

**ಪ್ರತಿ ಸಂಭವನೀಯ ಕನ್ನಡ ಪಠ್ಯ—ಈ ದಾಖಲೆ ಸೇರಿದಂತೆ—ಗ್ರಂಥಾಲಯದಲ್ಲಿ ಎಲ್ಲೋ ಅಸ್ತಿತ್ವದಲ್ಲಿದೆ.**
